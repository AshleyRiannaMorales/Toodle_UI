# Toodle
An app that helps you manage your tasks, appointments, and notes.

## Features
- Task Management 
- Calendar Overview
- Note-Taking

## Installation
### Prerequisites
- Install [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Ensure flutter doctor is error-free.
### Steps
1. Clone the repository
> git clone https://github.com/AshleyRiannaMorales/Toodle_UI.git

2. Access the downloaded repository
> cd todolist_ui

3. Install dependencies
> flutter pub get

4. Run the app
> flutter run
